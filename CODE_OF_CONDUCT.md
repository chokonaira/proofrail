# Code of Conduct

PermitRail is a technical project for building safer agent systems. Discussion
should stay direct, respectful, and grounded in evidence.

Expected behavior:

- Be precise and constructive.
- Critique code and ideas, not people.
- Disclose conflicts of interest when relevant.
- Respect security embargoes and private vulnerability reports.

Unacceptable behavior:

- harassment or targeted personal attacks
- public disclosure of private vulnerabilities before maintainers have had a
  reasonable chance to respond
- publishing secrets, private user data, or exploit details from another system
- impersonating maintainers or implying endorsement without permission

Maintainers may remove comments, close issues, block users, or restrict access
when needed to protect the project and its users.
